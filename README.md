A simple Tic-Tac-Toe game implemented using JavaScript, HTML, and CSS.


## Introduction

This project is a classic Tic-Tac-Toe game where two players take turns to mark spaces in a 3Ã—3 grid. The player who succeeds in placing three of their marks in a horizontal, vertical, or diagonal row wins the game.
